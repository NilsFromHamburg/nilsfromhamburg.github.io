This Website

This Website is based upon what I've learned from Andrei. Before that, I've had some very basic HTMl and CSS knowledge gut most of my designs were based on WordPress. I hope this is a thing of the past!

In the header you will find a nice background image that looks even nicer fixed creating this so called parallax effect. You may have also noticed some dotted lines and structure to this background image. I've actually overlapped multiple images here, some of them being transparent .png files which are repeated to create this pattern.

The main part of the website is made up of the main content and the main sidebar which just contains an image of me. I'll test some responsiveness features here soon.

The main content is made up of different sections changing their background color for layout purposes. Each section will contain a certain exercise from the course.

Finally there is the footer which contains some further links. You'll also notice the same pattern as picked for the header, this time on a single color background which makes it even more visible.
